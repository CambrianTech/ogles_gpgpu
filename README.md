# ogles_gpgpu - GPGPU for mobile devices and embedded systems using OpenGL ES 2.0

## Features

* fast and portable C++ code
* well documented
* contains several example applications

## Tested platforms

* iOS 7.1

## Tested devices

* Apple iPad 2
* Apple iPad 3
* Apple iPad 4
